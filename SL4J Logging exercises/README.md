# SLF4J Logging Exercises

This directory contains exercises for practicing and mastering logging in Java applications using SLF4J and Logback.

## Contents
* **[01 Logging Error Warning](./01_Logging_Error_Warning)** - Learning basic logging levels (INFO, WARN, ERROR) and configuration.
* **[02 Parameterized Logging](./02_Parameterized_Logging)** - Using parameterized logging messages to efficiently format logs.
* **[03 Using Different Appenders](./03_Using_Different_Appenders)** - Configuring console and file appenders in `logback.xml` to route logs to different destinations.
