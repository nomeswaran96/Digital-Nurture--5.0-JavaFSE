Welcome to my repository for the **Java Full Stack Engineering (FSE) Digital Nurture program**. This repository contains a comprehensive collection of hands-on exercises, assignments, and mini-projects spanning backend development, database management, architectural patterns, front-end development, and DevOps practices.

- **Backend Development:** Java SE, JUnit, Mockito, SLF4J (Logging)

## 🚀 How to Run the Projects

### Backend (Spring Boot & Java)
1. Navigate to any Spring project folder.
2. Run Maven compile:
   ```bash
   mvn clean install
   ```
3. Run the Spring Boot application:
   ```bash
   mvn spring-boot:run
   ```

### Frontend (React)
1. Navigate to the `Reactjs` folder.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run development server:
   ```bash
   npm start
   ```
